  2. Navigate to project: cd miniGames
  3. Option A - Automatic:
     • bash setup.sh
     • Server starts automatically
     • Open http://localhost:5000
  
  4. Option B - Manual:
     • python3 -m venv venv
     • source venv/bin/activate
     • pip install -r requirements.txt
     • python app.py
     • Open http://localhost:5000 in browser
