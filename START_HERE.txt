🎮 GAMES INCLUDED
================

1. ⭕ TIC TAC TOE
   • Play against an unbeatable AI
   • Win = 10 points, Draw = 5 points, Loss = 0 points
   • Minimax algorithm ensures optimal AI play

2. 🔢 NUMBER PUZZLE
   • Slide tiles to arrange numbers 1-15
   • Score based on efficiency (moves & time)
   • Up to 1,000 points possible

3. 🚗 CAR GAME
   • Collect coins with arrow keys or WASD
   • Avoid obstacles (3 hits = game over)
   • 60 seconds to score as many points as possible!

🔗 API ENDPOINTS
===============

POST /api/login
  └─ User login

POST /api/submit-score  
  └─ Submit game score

GET /api/leaderboard
  └─ Get top scores

GET /api/games/<game_name>/scores
  └─ Get all scores for a game

GET /api/games/<game_name>/user/<username>/scores
  └─ Get user's scores for a game

🔧 COMMON TASKS
===============

CHANGE COLORS:
  → Edit static/css/style.css
  → Search for :root {} section
  → Modify --primary-color, --secondary-color, etc.

ADD ANOTHER GAME:
  → Create new JS file in static/js/
  → Make it a class like existing games
  → Add route in app.py
  → Add card to dashboard.html

RESET DATABASE:
  → Delete minigames.db file
  → Restart the server

CHANGE PORT (e.g., 5001 instead of 5000):
  → Edit app.py, last line
  → Change app.run(port=5000) to app.run(port=5001)
  → Restart server


🐛 TROUBLESHOOTING
==================

Problem: "Address already in use"
  Solution: Change port in app.py or kill process on 5000

Problem: "Module not found"
  Solution: pip install -r requirements.txt

Problem: CSS/JS not loading
  Solution: Hard refresh (Ctrl+F5) or clear browser cache

Problem: Database error
  Solution: Delete minigames.db and restart server

Problem: Games not working
  Solution: Check browser console (F12) for errors

🎮 GAME SCORES EXPLAINED
=======================

TIC TAC TOE:
  Win   = 10 points ✅
  Draw  =  5 points 🤝
  Loss  =  0 points ❌

NUMBER PUZZLE:
  Score = 1000 - (moves × 5) - (seconds)
  Range: 100 to 1000 points
  Fewer moves = Higher score

CAR GAME:
  Per coin     = +10 points
  Per obstacle = -50 health
  Time bonus   = +2 per second remaining
  Game over when health = 0 or time = 0
